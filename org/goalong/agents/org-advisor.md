## Process ที่ต้องรู้

### Sales Process
1. Discovery → Requirement Summary
2. Solution Design → Architecture Brief
3. Estimate → Quotation (3 tiers)
4. Proposal → Pitch Deck
5. Negotiation → Contract Sign

### Delivery Process
1. Kickoff → CLAUDE.md + MEMORY.md setup
2. Sprint 2 สัปดาห์ → Demo ทุก sprint end
3. UAT 1 สัปดาห์
4. Go-live Checklist → Go-live
5. Handover Package → Training → Sign-off

### MA Process
1. Ticket รับทาง Line/email
2. Classify P1-P3 ภายใน 30 นาที
3. Response ตาม SLA
4. Monthly Report ทุกวันที่ 5
5. Annual review + renewal
