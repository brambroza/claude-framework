# Pitch Deck Skill

## หลักการสำหรับ SME ไทย
- ไม่เกิน 10 slides — เจ้าของ SME ไม่มีเวลา
- ทุก slide ต้องตอบคำถาม "แล้วไง? ดียังไงกับธุรกิจฉัน?"
- ใช้ตัวเลขจริง ไม่ใช่ buzzword
- เตรียม "one pager" สรุป 1 หน้าให้เอาไปดูทีหลัง

## 10-Slide Structure

### Slide 1: Hook (15 วินาที)
```
"[ชื่อธุรกิจลูกค้า] ใช้เวลา [X ชั่วโมง/วัน] กับ [ปัญหา]
เราช่วยลดเหลือ [X นาที]"

[ภาพ before/after หรือ metric]
```

### Slide 2: ปัญหาที่เจอ
```
3 ปัญหาหลักที่เราเข้าใจ:
1. [ปัญหาที่ 1 — เป็น pain จริงๆ]
2. [ปัญหาที่ 2]
3. [ปัญหาที่ 3]

[แสดงว่าเราทำการบ้านมา ไม่ใช่ pitch แบบ generic]
```

### Slide 3: Solution ของเรา
```
[ชื่อ product/solution]
[ประโยค 1 บรรทัดที่อธิบายว่าทำอะไร]

✅ [ประโยชน์ 1]
✅ [ประโยชน์ 2]
✅ [ประโยชน์ 3]

[Screenshot หรือ mockup ถ้ามี]
```

### Slide 4: Demo / How it works
```
3 steps ง่ายๆ:
1. [Step 1 — ง่ายมาก]
2. [Step 2]
3. [ผลลัพธ์ที่ได้]

[GIF หรือ screenshot flow]
```

### Slide 5: ผลลัพธ์จริง (Social Proof)
```
[ชื่อลูกค้า similar business]
"[quote สั้นๆ จากลูกค้า]"

ผลลัพธ์:
- [metric จริง เช่น ลดเวลา X%, เพิ่มยอดขาย X%]
- [metric จริง]
```

### Slide 6: Timeline
```
[Gantt chart แบบง่าย]

เดือน 1: [สิ่งที่ลูกค้าจะเห็น]
เดือน 2: [สิ่งที่ลูกค้าจะเห็น]
เดือน 3: 🚀 Go-live

[ระบุ milestone ที่ลูกค้ามีส่วนร่วม]
```

### Slide 7: ราคา
```
[Package ที่แนะนำ] — XXX,XXX บาท

รวมอยู่แล้ว:
✅ พัฒนาทั้งหมด
✅ Training
✅ 3 เดือน warranty
✅ Source code เป็นของคุณ

ชำระ 4 งวด — งวดแรก 30%
```

### Slide 8: ทำไมต้องเลือกเรา
```
[ไม่เกิน 3 ข้อ — ข้อที่แตกต่างจริงๆ]

1. ทำงาน [X] ปี เชี่ยวชาญ [industry]
2. ทีมไทย ดูแลหลัง go-live จริง
3. Fixed price — ไม่มีค่าใช้จ่ายเพิ่มเติม
```

### Slide 9: เกี่ยวกับเรา (สั้นมาก)
```
[ชื่อบริษัท] ก่อตั้ง [ปี]
[จำนวน project] projects | [จำนวน client] clients
[logo ลูกค้าที่ไว้แสดง]
```

### Slide 10: Next Steps
```
ขั้นตอนถัดไป:

1. ✅ ยืนยัน proposal — วันนี้
2. 📅 Workshop 1 วัน — สัปดาห์นี้
3. 📝 เซ็นสัญญา — ภายใน [วันที่]
4. 🚀 เริ่มงาน — [วันที่]

[ชื่อ] | [เบอร์] | Line: [ID]
```

## One Pager (สรุป 1 หน้า)
สร้างควบคู่กับ deck เสมอ — ลูกค้า SME เอาไปให้คนอื่นดูต่อ:
- ปัญหา + solution 3 บรรทัด
- ราคาและระยะเวลา
- สิ่งที่ได้รับ
- ติดต่อ
