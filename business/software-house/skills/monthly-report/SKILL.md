# Monthly MA Report Skill

## Input ที่ต้องการ
- เดือนและปีของ report
- Uptime data (จาก UptimeRobot หรือ monitoring tool)
- Incident log ของเดือนนั้น
- งานที่ทำในเดือนนี้
- แผนเดือนหน้า


## วิธีใช้อย่างมีประสิทธิภาพ
1. เก็บ incident log ตลอดเดือนใน `clients/[client]/memory/MEMORY.md`
2. ต้นเดือน invoke skill นี้ → ดึงข้อมูลจาก MEMORY.md
3. ใช้เวลา 10-15 นาทีแทน 2 ชั่วโมง
4. ส่งผ่าน Line หรือ email ทุกวันที่ 5
